## Before Asking for Help

1. Check the documentation and FAQ
2. Search existing issues for similar problems
3. Enable debug mode and check logs
4. Provide:
   - WordPress version
   - PHP version
   - Plugin version
   - Error messages/logs
   - Steps to reproduce
